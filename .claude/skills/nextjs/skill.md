# Next.js Development Skill

## Overview
This skill provides expertise in Next.js development, covering modern React framework patterns, server-side rendering, static site generation, API routes, and full-stack development practices.

## Capabilities
- Create and configure Next.js applications
- Implement routing with App Router and Pages Router
- Set up API routes and server-side rendering
- Configure static site generation and incremental static regeneration
- Handle data fetching strategies (getStaticProps, getServerSideProps, getStaticPaths)
- Implement authentication and authorization flows
- Optimize performance with code splitting and lazy loading
- Configure TypeScript with Next.js
- Handle image optimization and asset management
- Set up deployment configurations for various platforms (Vercel, Netlify, etc.)

## Best Practices
- Follow Next.js file-based routing conventions
- Use appropriate data fetching methods based on use case
- Implement proper error handling and loading states
- Leverage Next.js built-in optimizations
- Follow accessibility guidelines
- Use TypeScript for type safety
- Implement proper SEO practices with Head component
- Handle form submissions and validation

## Common Patterns
- Component-based architecture
- Server Components vs Client Components
- Data fetching patterns
- Form handling
- Authentication workflows
- API route patterns
- Styling approaches (CSS Modules, Tailwind, Styled Components)
- Testing strategies (Jest, Cypress, React Testing Library)

## Troubleshooting
- Debug common build issues
- Handle dynamic imports correctly
- Resolve hydration errors
- Fix common TypeScript issues
- Address performance bottlenecks
- Solve routing problems
- Handle CORS issues with API routes
