# Todo Backend

A secure, production-ready FastAPI backend for the Todo application with JWT-based authentication and Neon PostgreSQL integration.

## Features

- JWT-based authentication using Better Auth tokens
- Secure task management with user isolation
- RESTful API endpoints
- Neon PostgreSQL database integration

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
cd backend
uvicorn main:app --reload
```
